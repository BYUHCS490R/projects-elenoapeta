Web

How are the Internet and Web different?

The Internet is a worldwide network that connects computers, while the Web is collection of information store on the internet and access through Web browser

ASCII and Unicode

ASCII is a basic system that assigns numbers to letters and symbols in English, it help to the computer to easy store and process the text

What is UTF-8

UTF stands for Unicode Transformation Format-8-bits, is a text encoding that lets computers handle characters from many languages. It uses up to four bytes and works with both ASCII and Unicode

URL

URL stands for Uniform Resource Locator, URL is the web address that tells your brower where to find a specific page or file online

Domain Name

It's the easy to remember the name of the website, like google.com, this points to its actual location online

HTTP

HTTP is a system that lets the browser request and to load the web content from a server. This is how the website send data to your screen

HTTPS

HTTPS is a secure version of HTTP.

IP Address

IP Adress is a number that identifies a device on a network so it can send and receive data

DNS

Domain Name System turns website names into IP addresses so the browers can find and load the right site

W3C Recommendations

W3C are guidelines that set the standards for how web technologies should work to ensure consistency and to compatility across the internet
